# Contributing to MeshClaw

MeshClaw is MIT-licensed open infrastructure for offline agent collaboration.

High-impact areas for contributors:
- **Real Hardware Transports**: Implementing real BLE (`btleplug`) and LoRa (serialport) drivers.
- **Agent Intelligence**: Native inference runners (using `candle`, `llama-rs` or Ollama integration).
- **Core Protocol**: Optimizing GossipSub density and DHT routing for low-power devices.
- **Ecosystem**: SDKs for Python, Node.js, and browser-based agents.

No CLA required. All contributions stay MIT.
